# ❤️ Heart Disease Prediction

## 📌 Overview
This project predicts heart disease using machine learning.

## 📊 Dataset
- Contains patient health data (age, cholesterol, BP, etc.)
- Used for binary classification (disease / no disease)

## ⚙️ Implementation
- Data preprocessing
- Feature encoding
- Model training (Random Forest / Gradient Boosting)
- Evaluation using accuracy
